Messing about with susygrid (susy-test_1.scss)

This dev project has been set up for using Susy2 grid system, with Grunt and Bower.

So I've tried out flexbox, which would be great for use in smaller projects and modules and used sparingly for now (no universal support, and doesn't really replace a grid system.)

For the grid system I'm trying out the much talked about susygrid. From everything I've seen and heard so far, it feels like it's generally more used and loved in the front-end dev community.

Maybe later I could introduce flexbox elements into modules within the grid...?
