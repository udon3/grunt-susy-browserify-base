Settings for asymmetric layout

susy globals:

output: isolate,
columns: 1 2 3 2 1

note: the columns value is a sass list. It says: the layout consists of 5 base columns.

isolate fixes sub-pixel rounding bugs

$location mixin

isolate output mode, requires $location (first, last, at n) to be specified in the span() mixin.

This content container:

span(3 at 2)

= spans 3 columns - it has a context of (2 3 2) , and starts at 2nd base column ((1 *2 3 2* 1)

Nested elements

"It's all about the context": the content container also has an asymmetric grid, which is (2 3 2). So nested elements need this passed into the span mixin:

Nest One

span(1 of (2 3 2) first)

Nest Two

span(2 of (2 3 2) last)
